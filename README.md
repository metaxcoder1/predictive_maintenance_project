# Predictive Maintenance Project

This project predicts maintenance requirements in solar power systems using synthetic data.

## Project Structure
- **data/**: Contains solar_power_data.csv with synthetic data.
- **models/**: Trained model file.
- **scripts/**: Python scripts for generating data, training the model, and making predictions.
- **notebooks/**: Jupyter notebooks for data generation and main code.

## Usage
1. **Generate Data**: Run `scripts/generate_data.py` to create sample data.
2. **Train Model**: Execute `scripts/train_model.py` to train and save the model.
3. **Predict**: Use `scripts/predict.py` to make predictions.
